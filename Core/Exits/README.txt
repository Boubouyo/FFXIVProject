One line correspond to 1 exit.

The format is this :
COMMENT;FIRST_ROOM;SECONDROOM;

COMMENT : Put everything you want (except the $ sign) to comment what exit it is
FIRST_ROOM : The name of your first room (must be IDENTICAL to the name in Locations/)
SECOND_ROOM : Same but for the second room
; : is used as a delimitation between the arguments
